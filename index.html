Free Game Finder

A frontend take-home project that displays free-to-play games and allows users to sort the results.

Project Overview

This app displays free-to-play games with working graphics, sorting, cards, links, and a footer. It loads local game data first so it works inside VS Code, classroom sandboxes, and interview environments. The live API button is optional.
